My Skills

User Flow Chart

Creating user flow charts is one of my core skills, allowing me to map out the steps users take to complete tasks within a product. This visual representation helps in identifying potential pain points and optimizing the user journey for a seamless experience.

Storyboard

I use storyboarding to visualize user interactions and emotional responses throughout their journey. This technique helps in identifying and addressing potential issues early in the design process, ensuring a cohesive and engaging user experience.

Collaboration

Effective communication is vital in UX design, and I excel at collaborating with cross-functional teams. My ability to convey ideas clearly and listen to feedback ensures that projects stay aligned with user needs and stakeholder goals.

Wireframing

Wireframing is essential for quickly iterating on design concepts. I create low-fidelity wireframes to outline the structure and functionality of a product, enabling efficient feedback and refinement before moving to high-fidelity designs.

Tools Used Figma, FigJam, Trello, Google Slides, Visual Studio Code, Procreate
